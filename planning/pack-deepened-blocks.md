# activity pack — the deepened blocks (the +10 that makes it a programme, not a session)
### one per strand · the open-ended heart of each day

The deepened block is where the real learning lands. It is **never "do more"** — it's the same three-move shape on every strand, so the day feels coherent and the experience-book observation is consistent:

> **do it · notice it · say something about it**

| strand | block | the three moves |
|---|---|---|
| language | book'o'clock | **read** → **look at the bones** → **retell** |
| art | reference-artwork making | **make** → **look closely** → **talk about choices** |
| stem | imagine playground / wonder world | **build** → **test** → **explain** |

The "do it" uses your **existing at-building content** (the book, the medium, the build challenge). The moves and the **prompt sets** below are the new, scripted part. Run them word-for-word until they're yours.

---

## A · language — book'o'clock (deepened read-aloud) ~35 min
**skills:** storytelling & comprehension (primary) · listening · vocabulary

### move 1 · read it — *dialogic* (~15 min)
Read today's book (your 6-day arc). Read **with** the children, not at them — pause and use a prompt, take a few answers, **expand** the best one, then read on.
**dialogic prompt set (fixed — rotate through them):**
1. *"what do you think happens next?"* (predict)
2. *"why do you think she did that?"* (reason)
3. *"how is he feeling right now? how can you tell?"* (infer · emotion)
4. *"has that ever happened to you?"* (connect to self)
5. *"what would **you** do?"* (apply)
6. point to a new word: *"'enormous' — that means really, really big. show me enormous!"* (vocabulary)

### move 2 · look at the bones (~10 min)
Go back to 2–3 spreads. Talk about the **pictures** (*"what do you see? how did the artist show it was night?"*) and name the **story bones** on a 5-icon strip:
> **who** (character) · **where** (setting) · **what went wrong** (problem) · **what they did** (attempt) · **how it ended** (ending)

### move 3 · retell it (~10 min)
The children retell or *reshape* the story together. **retell-format set (fixed 4 — rotate by day of the arc):**
1. **puppet retell** — act it out with puppets/props
2. **change the ending** — "what if it ended differently?"
3. **from another side** — "tell it from the wolf's point of view"
4. **group chain** — each child adds the next part in order
👁 *observe window:* this is where the **storytelling north star** shows — mark focus children (story-bones in order = going strong).

## B · art — reference-artwork making (deepened) ~40 min
**skills:** creative expression (primary) · colour · fine motor

### move 1 · make (~25 min)
Set up the day's medium + technique, show the **reference** (not a sample to copy), then **step back**. Children make their **own** piece. Minimal talk; let them work.

### move 2 · look closely — a "gallery look" (~8 min)
A short gallery walk / hold-up. **gallery-look prompt set (fixed — rotate):**
1. *"what did you use to make this part?"*
2. *"how did you make that — show me?"*
3. *"what's different about everyone's? what's the same?"*
4. *"which bit are you most proud of?"*
> looking is a skill — this grows observation, the quiet foundation under all art.

### move 3 · talk about choices (~7 min)
Each child says **one thing about their own choice.** **prompt set (fixed):**
1. *"i made the ____ because…"*
2. *"i used my fingers / the brush because…"*
3. *"my favourite colour here is ____ — it makes me feel…"*
👁 *observe window:* the **creative-expression** rungs show here (explains a choice = going strong) — mark focus children.

## C · stem — imagine playground / wonder world (deepened) ~40 min
**skills:** curiosity · problem solving (primary) · scientific reasoning

### move 1 · build / explore (~22 min)
Set the build challenge or the day's investigation, then **step back. Four questions only — never the answer.** Children build/explore in small groups.
**the four questions (the *only* things you say):**
1. *"what are you trying to make / find out?"*
2. *"what's working? what's not?"*
3. *"what could you try?"*
4. *"what happened when you tried it?"*

### move 2 · test — "what if…?" (~10 min)
The science move: change one thing and watch. **what-if prompt set (fixed — rotate):**
1. *"what if you push it / roll it / make it taller?"*
2. *"what do you think will happen first — guess, then try."* (predict → test)
3. *"can you make it go faster / hold more / stand up?"*
4. *"try it two ways — which works better?"* (fair test, lightly)

### move 3 · explain (~8 min)
Each child/group shows their build and says **one thing they found.** **prompt set (fixed):**
1. *"show us — what did you make?"*
2. *"what did you find out?"*
3. *"why do you think it ____ (fell / rolled / worked)?"*
👁 *observe window:* **curiosity + problem-solving + scientific-reasoning** rungs show here (predicts then explains = going strong) — mark focus children.

---

## why the same three moves on every strand
Not just for tidiness: the **"do · notice · say"** shape means (1) the child learns the same thinking habit — *make/try something, look at it, put it into words* — across art, stem and language; (2) the teacher has the **same observe window** every day (the "say" move = where you mark the book); and (3) it's the seed of the **why-split infographic** — one programme, three doors, one rhythm.
